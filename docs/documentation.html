Cultural Heritage Acquisition and Digitisation Knowledge Graph (CHAD-KG)

The CHAD-KG is a knowledge graph containing descriptive metadata and digitisation paradata for a series of cultural heritage objects. These objects were initially displayed in a temporary museum exhibition dedicated to Ulisse Aldrovandi, an influential Italian Renaissance naturalist and botanist.

CHAD-AP Ontology

The structure of CHAD-KG is built upon the Cultural Heritage Acquisition and Digitisation Application Profile (CHAD-AP), an OWL ontology designed for describing cultural heritage objects, as well as the processes of acquisition and digitisation. The ontology enables structured and machine-actionable data, providing a framework for representing cultural heritage information.

Documentation for CHAD-AP can be found here: CHAD-AP Documentation.

Construction of the Knowledge Graph

CHAD-KG was constructed using the Morph-KGC extension, which utilizes YARRRML mapping files, a configuration file, and a set of user-defined functions. This setup generates RDF triples that are aligned with the CHAD-AP ontology from tabular data.

The software documentation for Morph-KGC can be accessed here: Morph-KGC Documentation.
